Morae

In the video the user first clicks on Getting Started, upon selecting their answers they click on submit. This takes them to a results page.

From results page the user then clicks on Tomatoes, then goes back to the home page by clicking on the icon.

From there they go to Fruit/Veg Index and click on Tomatoes again. They then type a search which takes them to a blank search result page thus concluding the first part of the video.

Some of the usability issues they reported were not knowing which picture represents which option for the getting started quiz. They were also annoyed at having to scroll all the way back up to top.

To fix this I added a yes or no text underneath the questions while also implementing a link at the bottom of the vegetable index to take the user back to the top.
